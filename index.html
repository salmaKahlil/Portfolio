Projects

Digit Analysis Engine

The engine receives an arbitrarily large stream of LARGE integers and a digit as inputs. For each integer, the engine does CPU-intensive computation looking for the number of occurances of this digit in a range of 0 to this LARGE integer.



Tools: Java, JUnit, Proto Buffer, gRPC, and Design Patterns, OOP, Git, and GitHub

Pipelined RISC-V processor

This project involves the development of a RISC-V processor adhering to the RV32I base integer instruction set specifications. The implementation employs a pipelined architecture with proper hazard handling across stages, addressing data and control hazards. A single memory, serving both data and instructions, is used and is designed to be single-ported and byte-addressable.



Tools: Verilog HDL, Vivado, and Nexys A7 kit

Search Engine

This project involves the development of a search engine algorithm using graphs. The engine displays the followings: page rankes, web pages scores, and keyword-based search results.



Tools: C++, Graph Algorithm, and CSV handling

Calculator using FBGA Board

The project involves the development of a calculator on a Basys 3 FPGA board using Verilog HDL. This calculator encompasses a wide range of mathematical functions, including addition, subtraction, multiplication, and division. Additionally, it features error-handling capabilities, such as identifying and managing division by zero scenarios while ensuring precise rounding.



Tools: FBGA programming and Verilog HDL

Tomasulo Simulator

This project aims to create an educational simulation of the Tomasulo algorithm, which is widely used in computer hardware for scheduling and executing instructions out of order. Developed in C++, the simulation provides an interactive environment for users to explore and grasp the intricacies of this algorithm.



Tools: C++, File Handling, and GUI

RV32IC Disassembler

This project involves developing a command-line RV32IC disassembler, which translates machine code into assembly language for the RISC-V architecture. The disassembler reads a binary file containing the machine code of a program's instructions, located at memory address 0x00000000. It decodes each machine code word into a valid RV32IC instruction and outputs the disassembled program using ABI register names.



Tools: C++, OOP, Git, and GitHub

GUI application

The project entails the development of a Graphical User Interface (GUI) application that allows users to manipulate the size, shape, and color of an icon using various controls. The GUI features a JSlider for adjusting the size of the icon dynamically. Additionally, there are buttons available for users to select different icon types, including Rectangle, Eclipse, and Triangle. Furthermore, three additional buttons enable users to change the color of the icon to Red, Blue, or Yellow



Tools: Java, Java Reflection, Model-View-Controller (MVC) Pattern, and OOP
